# 智鼎味来官网可视化内容管理后台——保姆级使用教程

本教程将手把手带您了解如何**启动网站服务**、**登录管理后台**、**修改各模块文案**以及**保存发布与注销**。

---

## 🚀 第一步：启动运行服务

在您开始使用后台前，请确保前端与后端服务均正常运行。

### 1. 后端服务 (持久化 API 代理)
* **默认状态**：后端服务已注册为 Mac 的 `launchctl` 开机自启动守护进程，默认在后台默默运行，监听 **`9876`** 端口。
* **手动重启/启动命令**（若服务异常或想手动启动）：
  打开终端，进入 `api-proxy` 目录执行以下命令：
  ```bash
  # 停止后台服务
  launchctl unload com.user.codex-proxy.plist
  # 重新启动并载入最新代码
  launchctl load com.user.codex-proxy.plist
  ```

### 2. 前端服务 (Vite 客户端)
* 打开终端，进入 `zdwl` 目录运行：
  ```bash
  npm run dev
  ```
* 启动成功后，在浏览器中访问：**`http://localhost:5173`** 即可打开官网。

---

## 🔑 第二步：登录可视化管理后台

我们在官网设计了隐蔽但操作流畅的管理员入口：

1. **寻找入口**：
   在网页的**右下角**，有一个半透明的**灰色齿轮悬浮按钮** ⚙️。
2. **拉起面板**：
   点击该齿轮按钮，屏幕右侧会平滑拉出高档磨砂质感的“网站内容可视化管理后台”抽屉。
3. **输入口令**：
   在密码输入框中输入管理员密码：
   * 🔑 **默认密码**：`admin`
4. **进入后台**：
   点击 **“进入后台”** 按钮或直接敲击回车键，即可解锁并进入文案编辑器仪表盘。

> [!TIP]
> **容灾（Fallback）机制**：即使后端 API 代理服务没有启动，只要输入 `admin` 密码，系统也会自动降级到**本地离线沙盒模式**，您依然可以随意修改文案，改动会自动保存在浏览器的本地缓存（LocalStorage）中。

---

## 📝 第三步：修改各模块文案内容

解锁成功后，您将看到井然有序的文案修改表单，支持以下四大核心板块的实时修改：

### 1. 全局导航栏配置 (Navbar)
* **品牌/Logo名称**：修改网页左上角霓虹等离子发光 Logo 循环展示的公司简称，例如：`智鼎味来`。

### 2. 主页巨幕 (Hero Section)
* **标题上方小字 (Eyebrow)**：对应首屏最顶部的细体小标题。
* **主标题 (Title)**：首屏的超大展示语，出厂默认为：`科技向膳 美味生活`。
* **副标题 (Subtitle)**：首屏的大段介绍语。
  * *排版亮点*：为了防止段尾单字折行，该文本框支持较长介绍。文本在渲染时，段首会自动缩进两个标准汉字距离，且带有公司简称的高亮提亮效果。

### 3. 创始人信息 (Founder Info)
* **姓名 / 核心头衔**：修改创始人“程华”的姓名及勋章副标题。
* **人物履历**：对应 Bento Grid（核心优势与创始人合并版块）右侧卡片中的生平简介。修改后，左侧的文字描述会自动与右侧的人脸照片左右分栏排布，避免文字遮挡人像。

### 4. 卡片核心业务配置 (Bento Grid)
* 分别对应 Bento Grid 板块的三个科技及业务卡片：
  * **卡片 1** (科技向膳 - AURA烹饪算法)
  * **卡片 2** (中餐标准化落地)
  * **卡片 3** (多端智能协同终端)
* 您可以在输入框和多行文本框中，任意更新这三张大卡片的标题和详细技术描述。

## 💾 第四步：保存发布与安全注销

修改完心仪的内容后，请按以下步骤应用并保存：

### 1. 保存并发布
* 点击面板最底部的蓝色 **“保存并发布”** 按钮。
* **状态反馈**：按钮会瞬时切换为不可点击的 `正在保存...` 状态，防止误触。
* **成功提示**：
  * **如果后端在线**：页面顶部会弹出带有绿色对勾的 **“保存并发布成功！”** 悬浮气泡，修改数据将**物理写入**服务器的 `site_data.json` 文件中，全网所有访问用户均会同步看到更新。
  * **如果后端离线**：顶部会弹出 **“本地保存成功 (持久化后端未连通)”** 悬浮气泡，改动只保存在您当前浏览器的本地缓存中。
* 提示消失后，抽屉面板会自动收起，整个网站页面将即时刷新应用最新的文案，无需手动刷新网页！

### 2. 取消修改
* 如果您中途决定放弃本次修改，点击底部的 **“取消”** 按钮或右上角的 **“X”** 按钮即可，此时文案会自动回滚为保存前的状态。

### 3. 安全注销 (Logout)
* 为防止他人误操作，当您完成编辑后，可以点击面板左下角的半透明红色 **“注销登录”** 按钮。
* 系统会立即销毁浏览器里的鉴权 Session Token，锁定后台，并在下一次点击齿轮时重新要求输入密码。
